# ArkadyJarvisMAX

Порт бота **ArkadyJarvis** с Telegram на российский мессенджер **MAX** (@id5408285792_1_bot).

Архитектура и функциональность идентичны оригиналу — отличается только транспортный слой (aiogram → [maxapi](https://github.com/max-messenger/max-botapi-python)). Детали в `CLAUDE.md`.

## Что умеет

- Авторизация через Bitrix24 по `@username` в MAX
- Создание встреч (Bitrix24 Calendar) с парсингом времени/участников
- Поиск свободных слотов по календарям коллег (5 рабочих дней)
- Создание задач в Jira по свободному описанию (AI переформулирует по шаблону)
- Создание лидов в Bitrix CRM из текста или голосового сообщения
- Генерация изображений (Gemini 3 Pro Image) с опцией редактирования фото
- Свободные вопросы Claude (ask_ai)
- Суммаризация групповых чатов за день (ежедневный отчёт в DM)
- Проверка договора по чек-листу (PDF/DOCX/TXT → Claude)
- AI-юрист «Цицерон» — консультация по российскому законодательству
- Сократ — расшифровка записи встречи → ревью → заготовка для аналитика
- Глафира — AI-офис-менеджер через OpenClaw (браузерный RPA)
- Анатолий — AI-рекрутёр, оценка кандидатов в Potok.io
- Среда: генератор мем-лягушки. Понедельник: конструктивистский постер.
- Webhooks для Bitrix24 (нотификации и рассылки)

## Быстрый старт

```bash
cp .env.example .env
# заполни как минимум BOT_TOKEN и Bitrix/Claude переменные

python3.11 -m venv .venv
source .venv/bin/activate
pip install -e .

uvicorn app.main:app --host 0.0.0.0 --port 8002
```

Или через Docker:

```bash
docker compose up --build
```
